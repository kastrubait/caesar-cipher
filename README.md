# caesar-cipher

> **Ð¡aesar-cipher** will help you encrypt a given string. This will be done by shifting all letters N positions to the right (encode) or to the left (decode).

#### Installation and Start
> 1. Clone the repository
    **git clone** https://github.com/kastrubait/caesar-cipher.git
> 2. Install node_modules:
   **npm install**
> 3. Go to the directory:
    **cd caesar-cipher**
> 4. Start:
    **node caesar_cipher_cli** `[options]`
